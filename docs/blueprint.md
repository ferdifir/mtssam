# **App Name**: MTs Sunan Ampel Mulyosari

## Core Features:

- Homepage Video Integration: Embed the video from https://youtu.be/Y5vf0iKpVIA?si=NJHC2R2HmX3ziOTJ to play in full screen.
- Principal's Welcome: Display a welcoming speech from the principal.
- Student Achievements: Showcase the school's student achievements.
- Teacher Profiles: Display photos of teachers with the subjects they teach.
- Activity Gallery: Present a gallery of school activities.
- School Motto CTA: Include a call-to-action section containing the school motto.
- PPDB Registration Button: Provide a button linking to the PPDB registration page.

## Style Guidelines:

- Primary color: Deep teal (#008080) to represent the Islamic school's identity and create a sense of trust.
- Background color: Light beige (#F5F5DC) to provide a warm, inviting feel.
- Accent color: Golden yellow (#FFD700) to highlight important elements like buttons and headings, evoking prestige.
- Headline font: 'Playfair', serif, for elegant headings. Body font: 'PT Sans', sans-serif, for easy-to-read paragraphs.
- Use Islamic-themed icons subtly to reinforce the school's identity.
- Ensure a clean, spacious layout that makes navigation intuitive for students, parents, and visitors.
- Incorporate smooth scrolling and subtle transitions to enhance the user experience.
